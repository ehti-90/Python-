#  Snake Game

A classic Snake Game built in **Python** using the built-in **turtle** graphics library. This project was created to practice Object-Oriented Programming (OOP) concepts such as classes, inheritance, and object interaction.

---

##  Features

- Classic snake movement
- Food spawning
- Scoreboard
- Snake grows after eating food
- Collision detection
- Keyboard controls
- Object-Oriented design

---

##  Project Structure

```
snake_game/
│── main.py          # Main game loop
│── snake.py         # Snake class and movement logic
│── food.py          # Food generation
│── scoreboard.py    # Score tracking and display
```

---

## Requirements

No external libraries are required.
The game uses Python's built-in `turtle` module.




## 🎮 Controls

| Key | Action |
|------|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |
